React Server Componentsがクライアントコンポーネントと異なる点は？
Answer
RSCはサーバー上でのみレンダリングされ、クライアントにJavaScriptバンドルが送信されない。データフェッチをコンポーネント内で直接行え、バンドルサイズを削減できる。

クライアントコンポーネントとは異なり、useState や useEffect などのReact Hooksは使用できないが、async/await を直接使用してデータベースやAPIにアクセスできる。
