# サブスクリプション管理アプリ UIトレンド調査

> 調査日: 2026-03-23
> 目的: SubRadarのUI設計に活用するためのベンチマーク調査

---

## 1. Rocket Money（旧Truebill）

**概要**: 米国最大級のサブスク管理・予算管理アプリ。Rocket Companiesが2021年に買収し2022年にリブランド。

| 項目 | 内容 |
|------|------|
| カラーパレット | 「Sunset」グラデーション（オレンジ〜ピンク）をブランドカラーに採用。リブランド後は暖色系グラデーションが定義的要素 |
| カードデザイン | フラットデザイン + カラーコード化されたカテゴリ。シンプルで整理されたレイアウト |
| ダッシュボード | 月次支出の円グラフ + カテゴリ別内訳が中心。カスタマイズ可能なウィジェット対応 |
| 月次サマリー | 月次支出のパイチャート + カテゴリ色分け + サブスク一覧 |
| 特徴的UI要素 | iOSウィジェット、請求ネゴシエーション機能UI、サブスク自動検出・キャンセルボタン |
| 人気の理由 | サブスク自動検出 + ワンクリックキャンセル機能が革新的。大手ブランド（Rocket）の信頼性。無料プランで基本機能が使える |

## 2. Bobby（iOS サブスク管理）

**概要**: Yummygum製のiOS専用サブスク管理アプリ。シンプルさとビジュアルの美しさで定評。

| 項目 | 内容 |
|------|------|
| カラーパレット | カスタマイズ可能なカラーテーマ。ライト/ダークモード対応。各サブスクのブランドカラーをカード背景に反映 |
| カードデザイン | **カード型UI**が核心。各サブスクを独立したカードで表示。カスタムアイコン＋カラーで視覚的に識別 |
| ダッシュボード | リスト形式 + 次回支払日・金額を前面に配置。シンプルなサマリーバー |
| 月次サマリー | 月次/年次コストを最上部に大きく表示。支払い期日カレンダービュー |
| 特徴的UI要素 | 豊富なアイコンライブラリ、複数ホーム画面アイコン、フォント選択、iCloudSync |
| 人気の理由 | **デザイン品質の高さ**が最大の差別化。情報密度を抑えた美しいUI。iOS HIG完全準拠で操作が直感的 |

---

## 3. Chronicle（Bill Organizer）

**概要**: 請求書・サブスク管理専門アプリ。iOS/macOS/iPadOS対応。「一つのことを極める」哲学。

| 項目 | 内容 |
|------|------|
| カラーパレット | システムカラー対応（iOS/macOSのアクセントカラーに追従）。Liquid Glass UI（最新iOS対応）を採用 |
| カードデザイン | リスト形式メイン。カテゴリアイコン（サービス別専用アイコン多数）が特徴的 |
| ダッシュボード | 支払い期日ベースのリスト + 残高ビュー。シンプルなグラフとレポート |
| 月次サマリー | 詳細レポートと支払い履歴。確認番号も記録可能 |
| 特徴的UI要素 | **インテリジェント支払い額予測**（変動請求書の自動推定）、支払い履歴データベース、iPad最適化レイアウト |
| 人気の理由 | 「請求書管理に特化」した潔さ。Apple HIG完全準拠の洗練されたネイティブUI。Setappにも収録 |

---

## 4. Copilot Money

**概要**: 2024年Apple Design Award ファイナリスト。AIによるスマートカテゴリ分類が革新的。

| 項目 | 内容 |
|------|------|
| カラーパレット | ダークパープル〜グラデーション系。エレガントで高級感のあるダークテーマ |
| カードデザイン | **Swift Charts活用**のモダンなグラフカード。アダプティブなビジュアライゼーション |
| ダッシュボード | Cash Flow（収入・支出・純収益）の3軸ビジュアライゼーション。SwiftUI製の高パフォーマンスUI |
| 月次サマリー | トレンド比較チャート + AI生成インサイト + 貯蓄目標進捗 |
| 特徴的UI要素 | Genmoji（カスタム絵文字）カテゴリ、Savings Goalsタブ、10,000以上の金融機関連携、Mac/iPad対応 |
| 人気の理由 | **AIが自動分類**するインテリジェンスの精度。Apple Design Awardファイナリストの洗練されたUIクオリティ。Swift Charts採用のネイティブ感 |

---

## 5. Spendee

**概要**: チェコ発のクロスプラットフォーム家計管理アプリ。ビジュアル分析に強み。

| 項目 | 内容 |
|------|------|
| カラーパレット | カラフルなカテゴリ別カラーシステム。各支出カテゴリをアイコン＋色で識別。ダークモード対応 |
| カードデザイン | ウォレット（口座）別カード表示。カラーグラデーションのカード背景 |
| ダッシュボード | **グラフ・チャート重視**。パイチャート、デュアルラインチャート、カレンダービュー |
| 月次サマリー | カテゴリ別支出割合のパイチャート + 月次比較グラフ |
| 特徴的UI要素 | AIレシートスキャナー、ラベルシステム、ウォレット分離（現金/口座）、予算設定UI |
| 人気の理由 | **美しいグラフ**による支出の可視化が直感的。複数通貨/ウォレット対応で海外ユーザーにも人気 |

---

## 6. Emma（Financial Super App）

**概要**: ロンドン発。160万ユーザー突破。「Financial Super App」を標榜するオールインワン家計管理。

| 項目 | 内容 |
|------|------|
| カラーパレット | グリーン系ブランドカラー（金融の安心感・成長を象徴）。カラフルなカテゴリシステム |
| カードデザイン | アカウントカード + カテゴリカード。色とアイコン（ピクトグラフ）でカスタマイズ可能 |
| ダッシュボード | 全口座統合ビュー + サブスク/コミット支出の分離表示 |
| 月次サマリー | 週次/月次レポート + 予算達成状況 + サブスク自動検出リスト |
| 特徴的UI要素 | **True Balance**（サブスク引落後の実質残高）、Fraud Detection UI、Smart Rules、SSL/生体認証 |
| 人気の理由 | 「コミット支出（サブスク）を分離表示」するTrue Balanceが独自価値。UK市場での銀行連携の深さ |

---

## 7. Subtrack

**概要**: プライバシーファーストのシンプルなサブスク管理アプリ。データをデバイスのみに保存。

| 項目 | 内容 |
|------|------|
| カラーパレット | ミニマルなシステムカラー。不要な装飾を排除した清潔感 |
| カードデザイン | シンプルなリスト形式。必要最小限の情報のみ表示 |
| ダッシュボード | 月次・年次コスト一覧 + 更新日アラート |
| 月次サマリー | 月次・年次合計コストのシンプル表示 |
| 特徴的UI要素 | **ローカルのみ保存**（プライバシー重視）、広告なし、カスタム周期（週/月/年/カスタム）対応 |
| 人気の理由 | 「シンプルさとプライバシー」を求めるユーザー層に刺さる。余計な機能を一切排除した潔さ |

## SubRadarに取り入れるべきUI要素 TOP 10

### 1. 🎨 ブランドカラーによるサービス識別カード（Bobby参考）
各サブスクのブランドカラーをカード背景に反映する。Netflix＝赤、Spotify＝緑など、視覚的に即認識できる設計。

### 2. 📊 Cash Flowスタイルの3軸ダッシュボード（Copilot Money参考）
「今月の支出合計 / 年換算コスト / 前月比」の3軸を常にトップに配置。Swift Charts相当のなめらかなアニメーションを実装。

### 3. 💰 True Balance方式の実質残高表示（Emma参考）
銀行残高からサブスク引落予定額を差し引いた「実質残高」を表示。ユーザーが「本当に使えるお金」を把握できる。

### 4. 🔔 更新日カウントダウン通知（Bobby + Chronicle参考）
次回更新日までの日数を各カードに表示。「3日後に更新」などのカウントダウンバッジ。期日前のプッシュ通知。

### 5. 📅 カレンダービュー（Spendee参考）
月カレンダー上にサブスク支払い日をドット表示。一目で「今月いつ何が引き落とされるか」がわかる。

### 6. 🤖 AIサブスク自動検出（Rocket Money参考）
銀行・クレカの取引履歴から定期支払いを自動検出する機能。手動入力の手間を省き、見落としを防ぐ。

### 7. 🌙 洗練されたダークモード（Chronicle + Copilot参考）
iOSのシステムカラーに追従する適応型ダークモード。単なる黒背景ではなく、階層感のある深みのある設計。

### 8. 📈 年間コスト可視化（複数アプリ共通）
「年間でいくら払っているか」を強調表示。月額だと安く感じるサービスも年額換算で実態を把握させる。これは行動変容につながる重要なUX。

### 9. 🏷️ カテゴリ別色分けシステム（Spendee + Emma参考）
エンタメ/仕事/健康/学習など、カテゴリ別にカラーコードを割り当て。フィルタリングとサマリー表示の両方に活用。

### 10. ⚡ マイクロインタラクション（Copilot + Rocket Money参考）
支払い完了時のチェックアニメーション、スワイプで削除/一時停止、スクロール時のパララックス効果など。「使っていて気持ちいい」体験を演出する細部の動き。

## フィンテックUI 2024-2025 大局トレンド

- **AIドリブンパーソナライゼーション**: 支出パターンからインサイト自動生成
- **モバイルファースト徹底**: 片手操作、サムフレンドリーなボタン配置
- **マイクロインタラクション重視**: 小さなアニメーションが信頼感と満足感を演出
- **ゲーミフィケーション**: 節約目標達成バッジ、進捗バー
- **生体認証デフォルト化**: Face ID/指紋によるシームレスなセキュリティ
- **ウェルネス視点**: 「節約できた額」を前向きに伝えるポジティブUX設計
